Набор персонала

Основная (нижняя) область формы обработки предназначена для работы с кандидатами и носит название Активные кандидаты. В полях вспомогательной (верхней) области Неразобранные письма отображается информация о письмах, пришедших по электронной почте на адреса с назначением Переписка с кандидатами. Если работа с электронной почтой не ведется, область с неразобранными сообщениями можно скрыть.

Активные кандидаты

Основная область формы Активные кандидаты состоит из нескольких частей:

  • В левой части отображается дерево кандидатов, отсортированных и сгруппированных по реквизиту кандидата. Реквизит, по которому группируется и сортируется список, выбирается пользователем. Такими реквизитами могут быть: дата события, состояние, наименование, вакансия, ответственный, подразделение, должность и группа кандидатов.
  • В правой части отображаются описание выбранного кандидата (поле Описание кандидата), список писем по кандидатов (верхняя часть поля Переписка по кандидату) и текст выбранного электронного письма (нижняя часть поля Переписка по кандидату). В поле Переписка по кандидату отображаются письма по всем электронным адресам, а не только по адресам с назначением Переписка с кандидатами, как в поле вспомогательной области Не разобранные письма.

Менять реквизиты кандидата можно непосредственно из описания кандидата с помощью гиперссылок, вызывающих нужные формы, а также с помощью команд меню Кандидаты и контекстного меню.

Кроме изменения реквизитов кандидаты доступны такие функции как:

  • Переписка по кандидату
  • Создание физического лица и сотрудника по кандидату
  • Закрытие кандидата

Неразобранные письма

Область работы с неразобранными письмами состоит из двух частей:

  • В левой части отображается дерево списка писем, отсортированных и сгруппированных по по дате или по отправителю, — в зависимости от выбора пользователя;
  • В правой части отображается текст выбранного электронного письма.

На основании письма можно сразу оформить кандидата, после чего письмо исчезнет из списка неразобранных и будет прикреплено к созданному кандидату.

Выбранное письмо можно отметить как нежелательное. Такое письмо будет перенесено в папку Нежелательные.

С помощью кнопок командной панели формы обработки можно открыть справочники Вакансии и Источники информации, а также вызвать обработку Кадровое планирование.
