Выгрузка данных в бухгалтерские программы

В бухгалтерскую программу передаются сведения об отражении в бухгалтерском и налоговом учете сумм начисленной заработной платы и соответствующих налогов и взносов, кассовые и банковские документы по выплате зарплаты, а также данные учета доходов сотрудников для целей построения регламентированной отчетности по НДФЛ, страховым взносам, ЕСН, формирования расчетных ведомостей по расчетам с ФСС.

Общие правила обмена данными

  1. Перед очередным формированием экспортного файла с данными учета начисленной заработной платы (или при внедрении конфигурации «Зарплата и Управление Персоналом» в эксплуатацию) необходимо в бухгалтерской программе сформировать файл с описанием объектов аналитического учета обработкой «Выгрузка данных в конфигурацию "Зарплата и управление персоналом"» .
  2. Сформированный в бухгалтерской программе файл с описанием объектов аналитического учета следует загрузить в информационную базу конфигурации «Зарплата и Управление Персоналом» при помощи универсальной обработки обмена данными. Для этого необходимо:
    · выбрать в меню "Обмен данными" (меню "Сервис") пункт "Универсальный обмен данными";
    · в появившемся диалоге на закладке "Загрузка данных" указать имя файла с описанием объектов аналитического учета;
    · нажать на кнопку "Выполнить".
  3. Если в используемой "бухгалтерской" аналитике произошли какие-либо изменения, тогда необходимо отредактировать способы отражения зарплаты в учете, используемые в конфигурации «Зарплата и Управление Персоналом» (см. меню "Учет зарплаты", интерфейс "Расчет зарплаты организаций").
  4. Сформировать и провести документы по отражению начисленной зарплаты в регламентированном учете за очередной расчетный месяц.
  5. Выгрузить в файл данные об отражении начисленной зарплаты в регламентированном учете настоящей обработкой. Подробнее см. здесь.
  6. Сформированный в конфигурации «Зарплата и Управление Персоналом» файл с переносимыми данными следует загрузить в ИБ бухгалтерской программы. Для этого также используется универсальная обработка обмена данными.
  7. При загрузке данных в бухгалтерскую программу синхронизация организаций производится по ИНН и КПП.

Переносимые данные

В бухгалтерскую программу могут быть выгружены следующие документы:

  • «Отражение зарплаты в регл учете» со сведениями об отражении в бухгалтерском и налоговом учете сумм начисленной заработной платы и соответствующих налогов и взносов;
  • «Корректировка учета по НДФЛ, страховым взносам и ЕСН» с данными учета доходов сотрудников для целей формирования регламентированной отчетности по НДФЛ, страховым взносам, ЕСН, а также для формирования расчетных ведомостей по расчетам с ФСС (4-ФСС и 4а-ФСС);
  • документы по выплате зарплаты:
    • «Зарплата к выплате организации»;
    • «Расходный кассовый ордер»;
    • «Приходный кассовый ордер»;
    • «Платежное поручение исходящее»;
    • «Депонирование организаций»;
    • «Списание депонентов в доходы организаций».

Важно!

Обмен с программой «Бухгалтерия предприятия» редакция 1.6.

За период с 2011 года доступна выгрузка только бухгалтерских проводок и кассовых документов.
За период по 2010 год доступна выгрузка бухгалтерских проводок, кассовых документов, данных учета по НДФЛ и страховым взносам, данных учета пособий по социальному страхованию.

Обмен с программами «Бухгалтерия предприятия КОРП», «Бухгалтерия автономного учреждения КОРП», «Бухгалтерия предприятия» редакция 2.0 и «Бухгалтерия автономного учреждения».
Данные учета по НДФЛ и страховым взносам и данные учета пособий по социальному страхованию могут быть загружены начиная с версии конфигурации 2.0.19.
Если в программе «Зарплата и управление персоналом» включена поддержка расчета взносов на доплату к пенсии в угольной промышленности, загрузка бухгалтерских проводок возможна начиная с версии 2.019.

Выгрузка данных в «1С:Бухгалтерию 7.7»

Общие правила обмена данными бухгалтерского и налогового учета

  1. Перед очередным формированием экспортного файла с данными учета начисленной заработной платы (или при внедрении конфигурации «Зарплата и Управление Персоналом» в эксплуатацию) необходимо в программе «1С:Бухгалтерия 7.7» сформировать файл с описанием объектов аналитического учета. Для этого предназначен пункт "Выгрузка данных в конфигурацию "Зарплата и упр-ие персоналом"" меню "Обмен данными" (см. меню "Сервис") программы «1С:Бухгалтерия 7.7».
  2. Сформированный в программе «1С:Бухгалтерия 7.7» файл с описанием объектов аналитического учета следует загрузить в информационную базу конфигурации «Зарплата и Управление Персоналом» при помощи универсальной обработки обмена данными. Для этого необходимо:
    · выбрать пункт "Обмен данными" меню "Сервис";
    · в появившемся диалоге на закладке "Загрузка данных" указать имя файла с описанием объектов аналитического учета;
    · нажать на кнопку "Выполнить".
  3. Если в используемой "бухгалтерской" аналитике произошли какие-либо изменения, тогда необходимо отредактировать способы отражения зарплаты в учете, используемые в конфигурации «Зарплата и Управление Персоналом» (см. меню "Учет зарплаты", интерфейс "Расчет зарплаты организаций").
  4. Сформировать и провести документы по отражению начисленной зарплаты в регламентированном учете за очередной расчетный месяц.
  5. Выгрузить в файл данные об отражении начисленной зарплаты в регламентированном учете настоящей обработкой. Подробнее см. здесь.
  6. Сформированный в конфигурации «Зарплата и Управление Персоналом» файл с данными об отражении начисленной зарплаты в регламентированном учете следует загрузить в программу «1С:Бухгалтерия 7.7». Для этого в программе «1С:Бухгалтерия 7.7» предназначен пункт "Загрузка данных из конфигурации "Зарплата и упр-ие персоналом"" меню "Обмен данными" (см. меню "Сервис").

Внимание! Средства для выгрузки данных об элементах аналитического учета и загрузки данных об отражении начисленной зарплаты включены в релиз 7.70.476 конфигурации «Бухгалтерский учет, редакция 4.5».

Переносимые данные

Данные переносятся из документов «Отражение зарплаты в регл учете» по указанной в обработке выгрузки организации, дата которых находится в указанном периоде.

При загрузке в программу «1С:Бухгалтерия 7.7» создаются:

  • ручная операция с отражением данных бухгалтерского учета оплаты труда и начисленных налогов;
  • документы «Расходы на оплату труда» с отражением расходов по заработной плате в налоговом учете;
  • документы «Начисление налогов» с отражением данных по начисленным налогам в налоговом учете.
