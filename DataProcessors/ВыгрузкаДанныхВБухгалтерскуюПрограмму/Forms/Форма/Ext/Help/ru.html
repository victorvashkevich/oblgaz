Настройка выгрузки данных в бухгалтерскую программу

В полях «Период с» – «по» укажите период, за который будет производиться выгрузка данных. В поле «Файл данных» укажите имя внешнего файла, в который будет производиться выгрузка данных. Кнопка «Выполнить» позволяет выгрузить данные в указанный в обработке файл.

Для бухгалтерских программ «Бухгалтерия предприятия» редакция 1.6,  «Бухгалтерия предприятия» редакция 2.0, «Бухгалтерия предприятия КОРП»,  «Бухгалтерия автономного учреждения КОРП» и «Бухгалтерия автономного учреждения» доступна настройка объема выгружаемых данных.

Выгрузка данных по отражению зарплаты в регламентированном учете. Должен быть взведен флажок «Отражение зарплаты в регл учете»; при этом будут выгружаться документы «Отражение зарплаты в регл учете», дата которых попадает в указанный период.

Выгрузка данных учета для целей исчисления НДФЛ, страховых взносов (ЕСН). Должен быть взведен флажок «Данные учета по НДФЛ и страховым взносам (ЕСН)», при этом  будут выгружены:

  • документы «Корректировка учета по НДФЛ, страховым взносам и ЕСН» с соответствующими данными учета, дата которых попадает в указанный период;
  • данные из независимых регистров сведений: «Доходы с предыдущего места работы (НДФЛ)», «Применение вычетов (НДФЛ)» и «Стандартные вычеты физических лиц по НДФЛ», «Стандартные вычеты физических лиц на детей»;
  • данные из регистров накоплений: «НДФЛ сведения о доходах», «НДФЛ расчеты с бюджетом», «НДФЛ имущественные вычеты», «Страховые взносы исчисленные», «Учет доходов для исчисления страховых взносов», «ЕСН Исчисленный», «ЕСН Сведения о доходах», введенные за указанный период.

Данные из регистров накопления будут перенесены в документ «Корректировка учета по НДФЛ, страховым взносам и ЕСН». Дополнительно, по данным движений регистра «НДФЛ имущественные вычеты» с видом движения «Приход» будет создаваться документ «Подтверждение права на имущественный вычет».

Важно! Данные учета доходов сотрудников для целей формирования регламентированной отчетности по НДФЛ, страховым взносам (ЕСН) могут быть загружены в ИБ «Бухгалтерия предприятия» редакция 1.6, начиная с версии конфигурации 1.6.29, «Бухгалтерия предприятия КОРП», «Бухгалтерия автономного учреждения КОРП»,  «Бухгалтерия предприятия» редакция 2.0 и «Бухгалтерия автономного учреждения» начиная с версии конфигурации 2.0.19.

Выгрузка данных учета для формирования расчетных ведомостей по расчетам с ФСС. Должен быть взведен флажок «Данные учета пособий по социальному страхованию», при этом  будут выгружены:

  • документы «Корректировка учета по НДФЛ, страховым взносам и ЕСН» с соответствующими данными учета, дата которых попадает в указанный период;
  • данные из регистров: «Заработки получателей пособия по уходу за ребенком до полутора лет», «Пособия по уходу за ребенком до полутора лет» и «Пособия социальному страхованию», введенные за указанный период.

Данные регистров будут перенесены в документ «Корректировка учета по НДФЛ, страховым взносам и ЕСН».

Важно! Данные для формирования расчетных ведомостей по расчетам с ФСС могут быть загружены в ИБ «Бухгалтерия предприятия» редакция 1.6, начиная с версии конфигурации 1.6.30, «Бухгалтерия предприятия КОРП», «Бухгалтерия автономного учреждения КОРП»,  «Бухгалтерия предприятия» редакция 2.0 и «Бухгалтерия автономного учреждения» начиная с версии конфигурации 2.0.19.

Выгрузка кассовых документов. Должен быть взведен флажок «Кассовые документы» и указан вариант выгрузки «Все документы» или «Только депонирование».
Для варианта «Все документы» будут выгружаться следующие документы: «Зарплата к выплате организации», «Приходный кассовый ордер», «Расходный кассовый ордер», «Платежное поручение исходящее», «Депонирование организаций», «Списание депонентов в доходы организаций».
Для варианта «Только депонирование» будут выгружаться следующие документы: «Зарплата к выплате организации», «Депонирование организаций», «Списание депонентов в доходы организаций».

Выгрузка сводных данных. В форме «Настройка программы» на закладке «Бухучет зарплаты» необходимо указать режим выгрузки данных по сотрудникам. Если будет выбран вариант  «Сводно по сотрудникам», то документы по отражению зарплаты в регламентированном учете и кассовые документы будут выгружаться сводными по сотрудникам. Перейти к форме  «Настройка программы»  можно по гиперссылке информационной надписи.

Важно! Выгрузка сводных данных не будет произведена, если за указанный период выгрузки введены документы «Расходный кассовый ордер» с видом операции «Выплата заработной платы сотруднику». В этом случае рекомендуется либо выгружать подробные данные, либо отменить проведение документов  «Расходный кассовый ордер» с указанным видом операции.
