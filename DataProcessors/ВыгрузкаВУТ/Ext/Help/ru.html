Выгрузка физических лиц в "Управление торговлей ред. 10.3"

Обработка выгружает данные о «физических лицах», в конфигурацию "Управление торговлей ред. 10.3".

Выгружать данные можно либо через файл обмена, либо непосредственно в информационную базу приемника:

  • Для выгрузки данных через файл необходимо указать имя файла, в который будут выгружаться данные;
  • Для выгрузки данных непосредственно в информационную базу приемника необходимо указать:
    • Тип информационной базы:
      • "На сервере 1С:Предприятия".
      • "На данном компьютере или на компьютере в локальной сети"
    • В зависимости от значения реквизита: "Тип информционной базы" "Имя сервера 1С:Предприятия" и "Тип информационной базу", либо "Каталог информационной базы"
    • Будет ли осуществляться "Аутентификация Windows"
    • Имя пользователя
    • Пароль

Также нужно указать тип учета, по которому оформляются физические лица на работу и при необходимости заполнить отборы.
