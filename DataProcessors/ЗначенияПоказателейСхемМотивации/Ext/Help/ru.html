Показатели для расчета зарплаты

Обработка "Показатели для расчета зарплаты" - это специализированный Помощник ввода значений показателей схем мотивации для расчета зарплаты. Форма позволяет определять значения постоянных и ежемесячных показателей на выбранный период.

При использовании в компании управленческого учета зарплаты в форме возможен ввод показателей в двух режимах: по структуре юридических лиц и по центрам ответственности. Ввод значений показателей осуществляется двумя способами с группировками: по сотрудникам и по показателям. Для ввода значений индивидуальных показателей следует установить "Данные по сотрудникам", для ввода показателей других видов - "Данные по организациям и подразделениям". Введенные значения будут записаны в систему и использованы при расчете заработной платы в выбранном периоде.

Команда Заполнить по плановым начислениям (удержаниям) позволяет заполнить строки для тех показателей, которые используются в плановых начислениях или удержаниях сотрудников.
