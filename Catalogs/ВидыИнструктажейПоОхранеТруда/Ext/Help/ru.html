Справочник «Виды инструктажей по охране труда»

Справочник "Виды инструктажей по охране труда" используется для хранения в системе сведений об инструктажах по охране труда.

Прохождение инструктажа регистрируется в системе документом "Инструктаж по охране труда".

Заполнение полей справочника. В соответствующих полях указываются:

  • Проводится однократно - при установке данного признака система не выполняет расчет даты очередного инструктажа;
  • Срок действия инструктажа - определяет периодичность обязательного прохождения инструктажа.

В табличной части справочника указываются должности, для работы на которых обязательно наличие  данного вида инструктажа.

Справочник содержит предопределенный элемент "Вводный инструктаж", который является обязательным для однократного прохождения всеми сотрудниками.
