Справочник «Классификатор специальностей по образованию»

Справочник используется при вводе данных об образовании работников организаций. 

Справочник может быть заполнен по данным Общегосударственного классификатора Республики Беларусь "Специальности и квалификации" (ОКСК). Для этого предназначена кнопка «Подбор из ОКСК» в форме списка. При этом открывается список классификатора, в котором можно найти, выбрать и занести в справочник нужную специальность с квалификацией.
