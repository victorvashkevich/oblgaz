Справочник "Группы пользователей"

В справочнике хранятся пользватели, обобщенные в группы.

Справочник используется в механизмах органичения прав доступа на уровне записей, и в механизме дополнительных прав пользователей.

Настраивать и обслуживать права доступа для каждого пользователя системы достаточно трудоемко, если число пользователей велико. Для упрощения решения таких задач пользователей системы группируют в группы. Назначение же прав доступа уже производится не для каждого пользователя в отдельности, а для групп пользователей.

Каждого пользователя можно включить в неограниченное число групп пользователей.
