Справочник "Группы почтовой рассылки"

Справочник предназначен для группировки объектов конфигурации ("Пользователи", "Организации" или "Физические лица") в группы рассылки.

Для каждой группы указывается ответственный за администрирование данной группы рассылки. Заполняется состав группы.

Указываются учетные записи, для которых данная группа рассылки будет доступна при подборе получателей из электронных писем. Группы рассылки можно отметить в списке на закладке «Использование в учетных записях». Доступны для выбора только те учетные записи, где ответственный за группу рассылки является администратором учетной записи.

При смене администратора в какой либо учетной записи, данная учетная запись исключается из доступных для тех групп, где ответственный за группу рассылки не совпадает с новым администратором учетной записи.
