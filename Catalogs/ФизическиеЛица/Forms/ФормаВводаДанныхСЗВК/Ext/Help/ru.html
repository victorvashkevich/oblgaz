Форма редактирования данных для СЗВ-К

Форма редактирования данных для СЗВ-К вызывается из формы редактирования данных о трудовой деятельности физлица (вызывается кнопкой «Труд» из основного диалога редактирования данных сотрудника).

Если ранее в этой форме уже была введена информация о предшествующих местах работы сотрудника, то часть данных формы СЗВ-К можно заполнить автоматически – кнопкой «Заполнить по трудовой деятельности».

Данные о проживании сотрудника в районах Крайнего Севера или приравненных к ним по состоянию на конец 2001 года указываются в полях «Территориальные условия» и «Районный коэффициент».

Данные о стаже

Все записи о периодах трудовой или деятельности сотрудника группируются по организациям, в которой работал сотрудник, и для каждой организации — по видам деятельности (работа, служба и др.), затем вносятся в таблицу сведений о стаже застрахованного лица. Для облегчения поиска нужных записей в таблице сведений о стаже застрахованного лица (когда их достаточно много) можно использовать список «Периоды трудовой или иной общественно полезной деятельности» – при позиционировании на любой строке из этого списка в таблице сведений о стаже застрахованного лица подсвечиваются серым цветом соответствующие записи.

Введенные записи о стаже можно расположить в хронологическом порядке, нажав кнопку «Расположить».

Сведения для оценки пенсионных прав

Итоговая таблица «Сведения для оценки пенсионных прав застрахованного лица» может быть заполнена автоматически по введенным записям о стаже (кнопка «Заполнить»), а также введена вручную.
