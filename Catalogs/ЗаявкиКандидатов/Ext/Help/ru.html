Справочник "Кандидаты"

Справочник предназначен для хранения кандидатов на вакантные должности.

 

Кандидат может характеризоваться следующими свойствами:

  • Дата поступления кандидата
  • Состояние кандидата - список определяемых пользователем состояний
  • Важность

 

Для кандидата можно указать, на какую должность и в какое подразделение претендует кандидат, а также ответственного по кандидату, выбрав его из справочника "Пользователи".

Кроме того, для кандидата можно указать Вакансию, на которую претендует кандидат. В этом случае, реквизиты Должность, Подразделение и Ответственный будут взяты из Вакансии.

 

Большое текстовое поле предназначено для ввода различной информации по кандидату в неструктурированной форме.

 

В качестве дополнительной информации можно указать:

  • Источник информации, из которого кандидат узнал о вакансии, либо источник, из которого Вы узнали о кандидате.
  • Ссылку на кандидата - элемент справочника "Физические лица".
  • Группу кандидатов, по которой можно ограничивать доступ к кандидатам.

 

Для работы с кандидатами предназначен инструмент "Набор персонала".

 

После того, как работа по кандидату завершена, можно установить одноименный признак в кандидате и написать результаты работы по кандидату.

Кандидаты с установленным признаком "Работа по кандидату завершена" не будут отображаться в инструменте "Набор персонала".
