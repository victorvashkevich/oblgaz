Документ «Прохождение обучения»

Документ предназначен для учета обучения на предприятии.

Учет пройденного обучения

Для учета пройденного обучения необходимо:

  • Выбрать тип обучения;
  • Выбрать работников, которые прошли или должны пройти обучения.  Для быстрого подбора работников предназначена кнопка "Подбор". Меню «Заполнить» также позволяет автоматически занести в табличную часть документа работников, отобранных по произвольным критериям;
  • Задать дату окончания курса;
  • Провести документ.

Результат пройденного обучения будет отражен в отчете Статистика кадров организаций.
