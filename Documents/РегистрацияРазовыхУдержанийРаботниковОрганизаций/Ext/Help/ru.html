Документ «Регистрация разовых удержаний работников организаций»

Документ предназначен для регистрации разовых удержаний работников организаций.

Заполнение полей документа. В соответствующих полях документа указываются:

  • Организация (по умолчанию это организация, указанная в настройках пользователя);
  • Ответственный (по умолчанию это ответственный, указанный в настройках пользователя);
  • Месяц начисления (по умолчанию текущий месяц) - определяет, в расчетном листке какого месяца будет отражено удержание;

В случае неоходимости загрузки в данный документ сумм за телефонные разговоры по корпоративному плану, необходимо нажать на кнопку «Загрузить», в открывшемся окне выбрать файл, из которого будет заполняться информация, выбрать соответствующий вид расчета для удержания и нажать кнопку «Заполнить». Таблица в форме будет автоматически заполнена данными из файла, которую можно при необходимости откорректировать. При нажати на кнопку «Выполнить» таблица с данными переносится в таблицу документа.
