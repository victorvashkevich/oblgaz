Документ «Сведения о заработке работников (ПУ-3)»

Используется в персонифицированном учете.

Используется для передачи данных по первоначальной регистрации или при изменении данных.

В табличную часть документа выбираются сведения из базы данных на дату документа. Все проверки на соответствие заполненных данных и необходимые изменения проводятся в программе ФСЗН "Ввод ДПУ".

 

При нажатии кнопки "ПолучитьФайл" на экран выводится файл пачки. Для сохранения файла необходимо выбрать меню "Файл/Сохранить как...". Выберите путь и укажите имя файла. Тип файла необходимо выбрать ANSI (Windows)(*.txt)  или OEM (DOS)(*.txt) в зависимости от Ваших настроек в программе "Ввод ДПУ".
