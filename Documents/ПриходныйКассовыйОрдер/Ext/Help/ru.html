Документ «Приходный кассовый ордер»

Документ «Приходный кассовый ордер» (ПКО) предназначен для отражения наличных денежных средств, поступающих в кассу от работников, во взаиморасчетах с работниками.

В зависимости от установленного вида операции с помощью данного документа может быть зафиксировано поступление наличных денежных средств по следующим операциям:

  • Расчеты по кредитам и займам с работниками;

  • Возврат денежных средств работником.

Расчеты по кредитам и займам с работниками. Операция предназначена для отражения погашения работником задолженности по договору займа. При оформлении даной операции необходимо указать работника, который погашает займ, и договор займа. При проведении сумма документа автоматически распределяется следующим образом: сначала производится погашение задолженности по процентам, оставшаяся сумма идет на погашение основной суммы займа.

Возврат денежных средств работником. Операция предназначена для отражения погашения работником задолженности перед организацией (предприятием) по оплате труда. При оформлении даной операции достаточно указать работника, вносящего задолженность, и сумму.

Автоматическое заполнение на основании других документов. Документ может быть введен на основании документа «Договор займа с работником», при этом вид операции выставляется "Расчеты по кредитам и займам с работниками", из договора займа переносится вид учета, а в качестве суммы заносится остаток непогашенной задолженности по займу, включая задолженность по процентам.

Печатные формы. После оформления документа и сохранения его в ИБ можно распечатать унифицированную форму КО-1.
