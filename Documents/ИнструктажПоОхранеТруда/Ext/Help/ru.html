Документ «Инструктаж по охране труда»

Документом «Инструктаж по охране труда» регистрируется факт прохождения сотрудниками инструктажей по охране труда.

Заполнение полей документа. В соответствующих полях документа указываются:

  • Организация (по умолчанию это организация, указанная в настройках пользователя);
  • Вид инструктажа - определяет вид проводимого инструктажа;
  • Ответственный (по умолчанию это ответственный, указанный в настройках пользователя).

В табличной части документа указываются сотрудники, прошедшие инструктаж.

Автоматическое заполнение на основании других документов. Документ может быть заполнен по виду инструктажа, при этом в табличную часть автоматически попадут сотрудники, работающие на должностях, для которых указанный вид инструктажа обязателен для прохождения при условии, что срок его действия истек, или инструктаж еще не был пройден.

Печатные формы. Из формы списка документов можно распечатать "Журнал учета инструктажей по охране труда".
