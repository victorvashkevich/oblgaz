Документ «Отражение зарплаты в управленческом учете»

Документ предназначен для регистрации затрат по центрам ответственности.

 

В документе предусмотрена возможность автоматического заполнения затрат в зависимости от выбранного метода планирования учета затрат:

  • Метод начислений
  • Кассовый метод

 

Метод начислений

Если для программы установлен метод начислений, затраты заполняются по начисленным в регламентированном и управленческом учете начислениях текущего месяца, а также по рассчитанным страховым взносам (до 2010 года ЕСН).

 

Кассовый метод

В этом случае документ заполняется в три шага:

1. Считаются выплаты по регламентированному и управленческому учету текущего месяца. К ним плюсуется рассчитанные страховые взносы (до 2010 года ЕСН) и НДФЛ прошлого месяца.

ВАЖНО! Вне зависимости от того, по каким организациям были выплаты, общая сумма выплат будет распределена пропорционально начислениям по всем организациям. Таким образом, даже если зарплата была выплачена только по одной организации, распределена она будет даже по тем организациям, для которых были зарегистрированы начисления, а выплат не производилось.

2. Пользователь проверяет корректность сумм и корректирует суммы при необходимости

3. Полученные суммы распределяются по затратам пропорционально начислениям прошлого месяца

Сумма по страховым взносам (до 2010 года ЕСН) и НДФЛ распределяется только по тем начислениям, у которых указан любой код дохода страховых взносов (до 2010 года ЕСН) и НДФЛ

 

Для обоих методов планирования учета затрат пользователь может указать, нужно или нет выделять налоги. Если пользователь выбрал выделение налогов, в документе появится еще одна колонка "Затрата", в которой будет указано, зарплата это или один из налогов.
