Документ «Регистрация нарушений по охране труда»

Документом «Регистрация нарушений по охране труда» регистрируются факты нарушения сотрудниками организаций законодательства по охране труда.

Заполнение полей документа. В соответствующих полях документа указываются:

  • Организация (по умолчанию это организация, указанная в настройках пользователя);
  • Ответственный (по умолчанию это ответственный, указанный в настройках пользователя);
  • Номер и дата приказа о наказании за нарушение

В табличной части «Работники организаций» документа указывается:

  • работник
  • вид нарушения (выбирается из соответствующего справочника)
  • дата нарушения;

В табличной части «Виды наказаний» документа указывается вид наказания, причем для одного сотрудника для каждого вида нарушения можно вводить несколько видов наказаний.

Автоматическое заполнение. Документ может быть автоматически заполнен работниками организации, отобранными по произвольным критериям; для этого предназначено меню «Заполнить» над табличной частью документа.

Проведение документа. При проведении документов программа проверяет, были ли работники, указанные в документе, приняты на работу и не уволены. При обнаружении таких ситуаций выдается сообщение об ошибке и документ не проводится.
