Документ "Корректировка записей регистров накопления"

Документ предназначен для ручной корректировки записей регистров накопления и для ввода остатков.

По кнопке «Настройка» верхней командной панели формы открывается форма настройки состава регистров. Флажками отмечаются те регистры, записи по которым планируется ввести.

На форме документа появляются закладки, соответствующие выбранным регистрам.

В табличные поля на закладках вводятся необходимые данные.
