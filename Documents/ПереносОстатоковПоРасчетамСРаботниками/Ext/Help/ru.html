Документ «Перенос остатков по расчетам с работниками»

Документ предназначет для переноса остатоко по взаиморасчетов с работниками организации. В поле Месяц Начисления указывается месяц на начало которого, расчитываеются остатки по взаиморасчетам с работниками.

Табличную часть документа можно заполнить автоматически через меню "Заполнить" кнопкой "По всем работникам организации".

Для ручного заполнения табличной части работники организации добавляются при помощи конопки "Добавить" или "Подбор". Для расчета остатков в табличной части документа заполненной в ручную используется меню "Заполнить" кнопка "По работникам".
