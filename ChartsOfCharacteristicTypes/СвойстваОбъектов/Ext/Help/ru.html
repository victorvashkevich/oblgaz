Заполнение видов и значений свойств объектов

Для каждого объекта могут быть назначены значения видов свойств различного типа. Значения видов свойств заносятся с помощью обработки "Свойства объектов".

Для свойства объектов могут быть назначены различные типы значения свойств.

На этапе конфигурирования в список свойств объектов могут быть добавлены предопределенные значения свойств, категорий или характеристик.

В процессе работы с типовой конфигурацией пользователь может добавлять свойства, имеющие различные типы значений. Добавление нового вида свойства производится в отдельном диалоговом окне, которое открывается при нажатии на клавишу INS или при выборе соответствующей пиктограммы на панели управления.

В соответствующих полях вводится информация о наименовании вводимого свойства; определяется справочник, для объектов которого предназначено данное свойство, и тип значений для данного свойства.

Диалоговое окно состоит из двух закладок:

  • На закладке "Назначение свойства" можно указать конкретные объекты или группы объектов, для которых будет предназначено данное свойство. Так, например, свойство "Коммуникабельность" целесообразно назначить для группе работников "Менеджеры".
    В список объектов, для которых предназначено свойство, можно добавить несколько групп справочника или даже указать конкретных физических лиц, если назначаемое свойство имеет смысл только для них. Если в список объектов, для которых предназначено свойство, не выбран ни один элемент, то данное свойство назначается всем объектам указанного справочника или документа.
  • На "Значения свойства" вводится список возможных значений для данного вида свойства.
