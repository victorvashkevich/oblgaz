Установка даты запрета изменения данных

В конфигурации поддерживается возможность установки ограничения изменения данных ранее определенной даты (далее "дата запрета").

Контроль ограничения изменения данных выполняется для:

  • всех документов при проведении и отмене проведения;
  • наборов записей регистров накопления, регистров расчета и регистров сведений, при этом контроль выполняется во всех случаях изменения записей регистров, включая программное.

Для документов проверяемым значением является дата этого документа, для наборов записей - период каждой из записей данного набора.

После указания даты запрета изменения данных документы за указанную или более раннюю дату будут доступны только для просмотра.

Дата запрета может устанавливаться как для всех пользователей, так и для каждого пользователя/группы пользователей в отдельности. В случае если для пользователя установлено несколько дат запрета (например, общая для управленческого учета, а также даты, установленные для групп, в которые входит пользователь) , в качестве ограничивающей даты берется минимальная из установленных дат, то есть та дата, которая наименее ограничивает пользователя.

Ограничение по дате запрета не распространяется на пользователей, которым доступна роль "Полные права".

Различаются понятия "основной даты запрета" и "даты ограничения изменения регламентной информации". Соответственно, в область действия основной даты попадают управленческие данные, вторая дата ограничивает изменение регламентной информации, раздельно для каждой из организаций. Если одновременно не задано значение и для основной даты и для регламентной - контроль изменения данных не будет задействован.

"Основная дата запрета" устанавливается на закладке "Основная дата" формы. Отображение заданных значений в разрезе пользователей/групп пользователей организовано в виде дерева, что позволяет задать значение как для общей для всех даты, так и для каждого пользователя/группы пользователей в отдельности. Для того, что бы установить для пользователя/группы пользователя "свою" дату запрета, необходимо добавить его/ее в дерево кнопкой "Добавить"

Установка ограничения на изменение регламентной информации выполняется аналогичным способом на закладке "Ограничение регламентной информации".
