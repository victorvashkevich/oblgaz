Настройка параметров учета

Настройка параметров учета вызывается в "Полном интерфейсе" из меню "Сервис" выбором пункта "Настройка параметров учета". В диалоговом окне присутствуют закладки, на которых задаются параметры, по которым будет вестись учет на предприятии.

Закладка "Валюты"

Валюта регламентированного учета — базовая валюта, курс которой всегда равен 1 (для России — рубли). Курсы всех других валют выражаются в единицах валюты регламентированного учета.
Валюта управленческого учета — валюта, используемая для оценки данных в управленческом учете. Обычно в качестве валюты управленческого учета указывают доллар США.

Закладка "Обмен данными"

Использовать механизм автообмена данными – установка флажка влючает использование автоматического обмена данными. Если в используется РИБ (распределенная информационная база), или другие варианты обменов с использованием планов обмена, то возможно настроить обмен данными без необходимости интерактивного запуска обмена пользователем. Все условия обменов, в том числе и автоматических, хранятся в справочнике "Настройки обмена данными".

Пользователь для регламентных заданий в файловом режиме – указывается пользователь, под которым будут выполняться автоматические обмены. При использовании клиент-серверной информационной базы нет необходимости в указании такого пользователя, потому что все обмены выполняются на сервере 1С:Предприятия, и даже если ни один пользователь не работает в системе, то автообмен все равно будет работать. Для файл-серверной информационной базы автообмен будет выполняться только в том случае, если указанный пользователь в момент наступления времени автообмена работает в системе.

Интервал опроса регламентных заданий в файловом режиме – интервал времени в секундах. С указанной периодичностью будет опрашиваться система на предмет понимания, нужно или нет запускать автообмен. Этот параметр так же необходимо указывать только для файл-серверных информационных баз (см прошлый пункт описания).

Закладка "Дополнительно"

Использовать ограничения прав доступа на уровне записей – установка этого флажка позволяет активизировать механизм детального ограничения прав доступа. При помощи этого механизма конкретному пользователю можно ограничить права доступа к некоторых физическим лицам, при этом данные по другим физическим лицам пользователь будет видеть. После установки этого флажка в форме списка и элемента справочника "Группы пользователей" станет доступна кнопка "Права", при нажатии на которую производится детальная настройка прав доступа.

Каталог дополнительной информации – в данном поле необходимо задать имя каталога, используемого средствами интернет-поддержки пользователей для хранения дополнительных файлов.

В группе Почтовый клиент выбирается, какой почтовый клент используется пользователями.

Закладка "Шаблоны телефонов"

На этой закладке можно ввести обычно используемые шаблоны номеров телефонов.
